# my-design-system
 My own React component library
 
🚀 Design System – Reusable React Component Library

A scalable and customizable Design System built from scratch, providing a shared library of reusable and accessible UI components for React applications. Inspired by modern libraries like Material UI and Chakra UI.

🔹 Key Features:

🎨 Design Tokens – Centralized styling (colors, typography, spacing) using tokens.json, converted into CSS variables
🧱 Reusable Components – 30+ accessible React components (Button, Input, Modal, Toast, etc.) with keyboard & screen reader support
📖 Storybook Integration – Interactive environment to visualize and test all component variants
✅ Automated Testing – High-quality components with 90%+ test coverage
🎭 Theming Support – Easily customizable themes for different brands using the same components
📦 npm Package – Published as a reusable package for easy installation and integration

💡 This project focuses on scalability, reusability, accessibility, and consistency across modern web applications.
